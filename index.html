Selected Publications

TNDC: Towards Reliable Label Pre-Correction via MLLM-Assisted Semantic Prototype Construction
To be submitted
Fangjiao Zhang, et al.
Proposed a novel framework leveraging Multimodal Large Language Models (MLLMs) to construct high-quality semantic prototypes. This method effectively addresses the challenge of label correction in extreme noise scenarios by refining semantic representations.
EVFeX: An Efficient Vertical Federated XGBoost Algorithm Based on Optimized Secure Matrix Multiplication
Fangjiao Zhang, Li Wang, Chang Cui, et al.
Signal Processing, Vol 227, 109686, 2025. Developed a secure XGBoost framework that eliminates time-consuming homomorphic encryption by using optimized secure matrix multiplication. The algorithm achieves security equivalent to encryption while significantly enhancing computational efficiency for large-scale datasets.
Secure Vertical Federated Learning Based on Feature Disentanglement
Fangjiao Zhang, Guoqiang Li, Zhufeng Suo, et al.
Signal Processing, Vol 211, 109077, 2023. Proposed SVFL, a vertical federated learning framework inspired by feature disentanglement. It defends against label inference attacks by separating private features from shared representations, thereby reducing privacy leakage risks.
XMAM: X-Raying Models with a Matrix to Reveal Backdoor Attacks for Federated Learning
Jianyi Zhang*, Fangjiao Zhang* (Co-first Author), et al.
Digital Communications and Networks, 2023. Designed a novel aggregation method that "X-rays" local model updates using a probing matrix to reveal hidden backdoor patterns in the Softmax layer. This approach effectively identifies and filters out malicious updates from backdoor attackers.
